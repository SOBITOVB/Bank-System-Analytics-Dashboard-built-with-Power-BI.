🏦 Excited to share my latest project — a fully interactive Bank System Analytics Dashboard built with Power BI.

This dashboard was designed to provide deep insights into banking operations, customer behavior, branch performance, loan portfolio analytics, and operational efficiency through modern data visualization techniques.

One of the most interesting parts of this project was generating realistic banking datasets using Python and the Faker library, which helped simulate real-world banking transactions, customers, branches, loan records, and payment activities.

Key Features:
• Executive KPI Monitoring
• Customer Analytics & Regional Insights
• Branch & Regional Performance Tracking
• Loan Portfolio & Payment Analysis
• Operational Efficiency Monitoring
• Interactive Navigation & Drill-through Pages
• Shape Map & Geospatial Analytics
• Dynamic Filters and Cross-Filtering
